Язы́к программи́рования — формальный язык, предназначенный для записи компьютерных программ. Язык программирования определяет набор лексических, синтаксических и семантических правил, определяющих внешний вид программы и действия, которые выполнит исполнитель под её управлением. На данный момент языков программирования 253 языка на основе данных из таких источников, как GitHub и TIOBE (наиболее популярные языки программирования). Их количество начали отслеживать еще в 1991 году.
Всякий язык программирования имеет три основные составляющие: алфавит, синтаксис и семантику. Синтаксис - набор правил построения фраз алгоритмического языка, позволяющий определить, осмысленные предложения в этом языке. Смысловое содержание языковой конструкции называется семантикой.
Классификация
Языки программирования часто классифицируют по уровню. Уровень языка показывает, насколько язык близок к естественной для человека записи. По этому критерию можно выделить следующие уровни языков программирования:
  • машинные
  • машинно-оpиентиpованные (ассемблеpы)
  • машинно-независимые (языки высокого уровня).
